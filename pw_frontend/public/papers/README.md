# Papers Directory

Place your downloadable files here. Files in the `public` folder are served at the root URL in production.

## Required Files for Heroin Addiction Paper:

1. **PDF Paper**: `treating-heroin-addiction-paper.pdf`
   - Extract from: `src/assets/styles/papers/Treating Heroin Addiction Sustaining Long-Term Abstinence.pdf.zip`
   - Rename to: `treating-heroin-addiction-paper.pdf`

2. **PowerPoint Presentation**: `treating-heroin-addiction-presentation.pptx`
   - Add your Canva presentation here
   - Export from Canva as PPTX format
   - Name it: `treating-heroin-addiction-presentation.pptx`

## File Access

Files placed here will be accessible at:
- `/papers/treating-heroin-addiction-paper.pdf`
- `/papers/treating-heroin-addiction-presentation.pptx`

These paths work in both development and production.
